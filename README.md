# Faith + Build

A men's group building faith into products, services, and community.
Wednesdays · 6:30 PM · 7301 Sitio Salvia, Carlsbad.

> "Faith by itself, if it does not have works, is dead." — James 2:17

A static GitHub Pages site: a one-page landing site with a sign-up form and a faith /
Bible-study survey (both powered by Google Forms).

## Files
- `index.html` — the whole site (one page, anchor-linked sections)
- `styles.css` — design system (warm editorial + builder energy; colors in `:root`)
- `favicon.svg` — flame + "plus" mark
- `SETUP.md` — **start here**: create the two Google Forms, paste the links, publish

## Quick start
1. Read `SETUP.md`.
2. Create the two Google Forms (questions are written out, ready to paste).
3. Paste the form links into the `window.FORMS` block at the top of `index.html`.
4. Push to GitHub and turn on Pages.

Until the form links are set, every button falls back to a pre-filled email to
`soren@resonancepartners.us`, so the site is fully usable from minute one.
